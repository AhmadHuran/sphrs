# sphrs (WIP)

[Documentation](https://stefan-k.github.io/sphrs/sphrs/).


## Usage

Add this to your `Cargo.toml`:

```
[dependencies]
sphrs = { git = "https://github.com/stefan-k/sphrs.git" }
```
